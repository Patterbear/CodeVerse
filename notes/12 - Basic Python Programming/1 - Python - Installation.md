# Installation

# Windows

To install Python in Windows, go to this location: [http://python.org/download/](http://python.org/download/) and install the Python 3 Windows x86 MSI Installer.

https://youtu.be/4Mf0h3HphEA

In order to run Python from the command line, you then follow the instructions here: [http://docs.python.org/faq/windows.html#how-do-i-run-a-python-program-under-windows](http://docs.python.org/faq/windows.html#how-do-i-run-a-python-program-under-windows)

# Ubuntu

Ubuntu users must see Python installed by default.

You can run Python from the command line as follows:

`$ python <python-file>`
